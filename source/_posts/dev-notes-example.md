这是一篇使用"开发笔记"分类的示例文章。

## 说明

这篇文章演示了如何使用"开发笔记"分类，该分类与"技术"分类处于同一层级。

## 使用方法

在文章的 front matter 中使用以下格式：

```yaml
categories:
    - 开发笔记
```

## 示例代码

```python
def hello_world():
    print("Hello, 开发笔记!")
```

## 注意事项

- "开发笔记"是一个顶级分类，与"技术"同级
- 如果需要创建子分类，可以使用：
  ```yaml
  categories:
      - 开发笔记
      - 子分类名称
  ```
